## Project Description

A web-based template generator that demonstrates HTML5 Canvas API, JavaScript form handling, and CSS styling techniques through the creation of a Colorado ID template.

## Features

- Interactive form inputs
- Real-time canvas rendering
- Photo upload functionality
- Front/back template views
- Barcode generation using bwip-js
- Download as PNG

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- HTML5 Canvas API
- bwip-js library for barcode generation

## Installation

1. Clone this repository
2. Open `index.html` in a web browser
3. No server or build process required
